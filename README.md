## Installation
Do the following steps for installation

<br>
<br>

In the root folder create `.env` file 
```bash
touch .env
```

<br>

Put the followings in the env file

```text
NODE_ENV = development
PORT = 5000
MONGO_URI = <YOUR_MONGODB_URI>
JWT_SECRET = <YOUR_JWT_SECRET>
```

<br>

Install dependencies
```bash
npm i
```

<br>

Run the app
```bash
npm run dev
```
